# 🃏 Whot Card Game

Juego de cartas Whot para un jugador contra la CPU, desarrollado en HTML, CSS y JavaScript puro. No requiere instalación ni dependencias.

## 🎮 Jugar en línea

👉 **https://gasvel95.github.io/whot-game/**

## 🃏 Reglas básicas

- Haz coincidir la figura o el número con la carta del tope de la pila.
- Las cartas especiales tienen efectos únicos.

### Cartas especiales

| Carta | Efecto |
|-------|--------|
| **WHOT (20)** | Comodín: eliges la figura que sigue |
| **Hold On (1)** | Vuelves a jugar tú |
| **Pick Two (5)** | El oponente roba 2 cartas |
| **Suspension (8)** | El oponente pierde su turno |
| **General Market (14)** | El oponente roba 1 carta |

## 🚀 Activar GitHub Pages

1. Ve a **Settings → Pages**
2. En **Source**, selecciona `main` y carpeta `/ (root)`
3. Haz clic en **Save**
4. En unos minutos el juego estará disponible en `https://gasvel95.github.io/whot-game/`

## 🛠️ Ejecutar localmente

Simplemente abre el archivo `index.html` en tu navegador.
